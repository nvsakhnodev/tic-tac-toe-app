# 🎮 Tic-Tac-Toe App

Игра Крестики-нолики, созданная с помощью **HTML, SCSS и JavaScript**.

---

## Возможности

* Игровое поле 3x3
* Ходы по очереди (X / O)
* Определение победителя
* Подсветка выигрышной линии
* Подсчёт очков
* Кнопка "Начать заново"
* Кнопка "Сбросить счёт"
* Плавные анимации X и O

---

## Технологии

* HTML5
* SCSS / CSS3
* JavaScript
